# AgentForge Studio Frontend

This directory will contain the React frontend application for AgentForge Studio.

## Planned Stack

- React 18+
- TypeScript
- Tailwind CSS
- Vite (build tool)
- WebSocket client for real-time updates

## Features (Planned)

- Chat interface for communicating with the Intermediator agent
- Project dashboard showing all active projects
- Real-time agent status visualization
- Live preview of generated websites
- File browser for project files
- Git history viewer

## Development

Frontend development will begin after the backend is complete.

```bash
# Future commands
cd frontend
npm install
npm run dev
```
